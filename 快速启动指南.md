# 🚀 快速启动指南

## 第一步：等待依赖安装完成

当前正在安装 `snownlp`（情感分析库），请耐心等待下载完成。

下载进度可以在终端中看到，预计需要5-10分钟。

## 第二步：测试情感分析（可选）

当snownlp安装完成后，可以运行以下命令测试情感分析效果：

```bash
python test_sentiment.py
```

这会显示前5条新闻的情感分析示例，包括：
- 新闻标题
- 情感得分（0-1之间）
- 情感类别（正面/中性/负面）

## 第三步：启动应用

### 方法1：使用批处理文件（推荐）
双击运行：
```
启动应用.bat
```

### 方法2：命令行运行
```bash
streamlit run app.py
```

## 第四步：访问应用

浏览器会自动打开 http://localhost:8501

如果未自动打开，请手动在浏览器中访问该地址。

## 📊 首次加载说明

**重要提示：** 首次运行时，系统会：

1. **加载CSV数据**（约1秒）
2. **检测是否有sentiment列**
3. **如果没有，计算所有6702条新闻的情感得分**
   - 先显示前5条作为示例
   - 然后批量计算剩余新闻
   - 预计需要3-5分钟
4. **保存结果到CSV**
5. **显示主界面**

**后续加载会非常快**（直接从CSV读取情感得分）

## 🐛 常见问题

**Q: 页面加载很慢？**
A: 首次加载需要计算情感分析，之后会很快

**Q: 词云中文显示为方框？**
A: 系统会自动检测中文字体，一般不会出现此问题

**Q: 搜索没有结果？**
A: 检查关键词拼写，搜索不区分大小写

**Q: 如何更新数据？**
A: 点击侧边栏的"从官网更新数据"按钮
