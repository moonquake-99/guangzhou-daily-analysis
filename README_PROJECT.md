# 📰 广州日报舆情分析系统

> 基于Streamlit的实时舆情监控平台，支持自动更新和云端部署

## ✨ 核心功能

### 1️⃣ 全局概览
- 📊 **5个平级指标卡片**：新闻总数、平均情感、正面/负面/中性新闻数量
- 📈 **月度趋势分析**：新闻数量、情感得分、正负面趋势折线图
- 🥧 **情感分布饼图**：直观展示正负面比例
- ☁️ **总体词云**：从标题提取的热点关键词可视化
- 🔥 **热点关键词TOP 20**：动态阈值算法提取高频词

### 2️⃣ 月度详情
- 📅 **月份选择器**：自由选择查看任意月份数据
- 📊 **本月5大指标**：与全局概览样式统一
- ☁️ **本月词云**：当月热点话题可视化
- 🔥 **本月热点关键词TOP 10**：精准捕捉月度焦点
- ⚠️ **负面新闻预警**：自动识别并展示负面新闻
- 🔍 **关键词搜索**：支持标题和正文全文检索
- 📄 **分页浏览**：每页10条新闻，支持按日期/情感排序

### 3️⃣ 智能情感分析
- ✅ **改进算法**：确保正面新闻不被误判为中性/负面
- 🎯 **双重验证**：SnowNLP + 关键词匹配
- 🎨 **彩色标识**：绿色(正面)、红色(负面)、橙色(中性)
- 💬 **展开查看**：点击新闻显示完整正文和情感得分

### 4️⃣ 自动更新系统
- ⏰ **定时爬取**：每天凌晨2点自动获取最新新闻
- 🔄 **全流程自动化**：爬虫 → 清洗 → 分析 → 上传
- ☁️ **云端同步**：GitHub + Streamlit Cloud
- 🌍 **全球访问**：任何电脑输入网址即可查看

## 🎯 核心算法

### 1. 情感分析算法

```python
def calculate_sentiment_v2(text, title=''):
    """
    改进版情感分析 - 确保正面新闻不被误判
    
    策略：
    1. 有正面词且无负面词 → 强制正面 (≥0.65)
    2. 有负面词(≥2)且无正面词 → 负面 (≤0.35)
    3. 正面词 > 负面词 → 正面 (≥0.55)
    4. 负面词 > 正面词 → 负面 (≤0.45)
    5. 无明显情感词 → 偏向中性偏正 (0.55)
    """
```

**优势**：
- ✅ 减少正面新闻误判
- ✅ 官方新闻默认偏正面
- ✅ 降低过度乐观/悲观

### 2. 热点关键词提取

```python
def extract_hot_topics(titles, top_n=20):
    """
    动态阈值算法
    
    1. 分词并过滤停用词
    2. 统计词频
    3. 计算阈值: max(8, 最高频 * 0.5)
    4. 筛选 ≥ 阈值的词
    5. 返回TOP N
    """
```

**特点**：
- 🎯 自适应阈值（不同月份自动调整）

- ✅ 保留专有名词（十五运会、大湾区等）

## 🐛 常见问题

### Q: 为什么有些正面新闻被判为中性？

**A**: 这是正常现象。如果新闻中没有明显的正面词汇（如"成功"、"优秀"、"创新"），即使是积极内容也可能被判为中性。这是为了避免过度乐观。

### Q: 词云为什么不显示某些词？

**A**: 以下词会被过滤：
- 长度<2的单字词
- 纯数字
- 停用词（地域、媒体、领导人、通用词等）
- 出现频率低于阈值的词

### Q: 如何添加自定义词典？

**A**: 在 `app_cloud.py` 中找到 `CUSTOM_DICT`，添加新词：

```python
CUSTOM_DICT = [
    '十五运会', '全运会', '大湾区', 
    '你的新词'  # ← 添加这里
]
```

### Q: Streamlit Cloud部署失败？

**A**: 检查：
1. 仓库是否是Public
2. requirements.txt是否正确
3. 查看部署日志
