# 📁 项目文件清单

## ✅ 核心应用文件

### 1. app.py (643行)
**Streamlit主应用**
- 完整的情感分析仪表盘
- 8大功能模块
- 自动检测中文字体
- 279个停用词表
- 50+自定义词典
- 性能优化缓存

### 2. news_with_keywords.csv
**数据文件**（已有）
- 6,702条新闻数据
- 包含url, title, content, date, month, keywords列
- 首次运行时会自动添加sentiment列

## 🔧 辅助脚本

### 3. gzdaily_spider.py
**新闻爬虫**（已有）
- 多版面爬取
- JSON格式存储
- 支持增量更新

### 4. extract_hot_topics.py
**热点话题提取**（已有）
- 每月Top关键词
- 动态数量筛选
- 停用词过滤

### 5. extract_keywords_for_each_news.py
**单条新闻关键词提取**（已有）
- 为每条新闻提取15-20个关键词
- 保存到keywords列

### 6. test_sentiment.py
**情感分析测试脚本**
- 测试前5条新闻的情感得分
- 显示示例结果

## 💡 使用流程

1. **首次使用**
   ```
   双击 启动应用.bat
   → 等待依赖安装
   → 等待情感分析计算（3-5分钟）
   → 浏览器自动打开
   → 开始使用！
   ```

2. **日常使用**
   ```
   双击 启动应用.bat
   → 直接加载（秒开）
   → 选择月份查看分析
   → 搜索感兴趣的新闻
   ```

3. **数据更新**
   ```
   在应用中点击"从官网更新数据"
   → 运行爬虫获取新数据
   → 手动合并到CSV
   → 刷新页面
   ```

## ✨ 特色功能

1. **智能情感分析** - SnowNLP实时计算，自动缓存
2. **动态关键词** - 基于频次自动筛选，非固定数量
3. **精美词云** - 自动检测中文字体，美观大方
4. **负面预警** - Top 3负面新闻，及时发现风险
5. **实时搜索** - 标题关键词搜索，快速定位
6. **趋势分析** - 月度数量变化，一目了然
7. **数据更新** - 一键爬取最新新闻
8. **响应式设计** - 适配各种屏幕尺寸
