# 广州日报舆情分析系统

## 📋 项目简介

基于Streamlit构建的交互式舆情分析仪表盘，用于展示和分析广州日报新闻数据。

## ✨ 功能特性

1. **📊 月度数据分析**
   - 新闻数量统计
   - 平均情感得分
   - 正面/负面新闻数量

2. **😊 情感分析**
   - 使用SnowNLP计算每条新闻的情感得分
   - 总体情感分布饼图
   - 月度平均情感得分

3. **🔑 热点话题追踪**
   - 每月Top 15关键词提取
   - 基于标题分词和停用词过滤
   - 水平条形图展示

4. **☁️ 词云可视化**
   - 整体词云（所有月份）
   - 单月词云
   - 自动检测中文字体

5. **⚠️ 负面新闻预警**
   - 展示每月情感得分最低的3条新闻
   - 可展开查看原文

6. **🔍 新闻搜索**
   - 按关键词搜索标题
   - 实时显示搜索结果
   - 点击查看正文

7. **📈 趋势分析**
   - 月度新闻数量趋势图
   - 柱状图+折线图组合

8. **🔄 数据更新**
   - 一键从官网爬取最新数据
   - 自动合并到现有数据

## 🛠️ 安装依赖

```bash
pip install streamlit pandas matplotlib wordcloud jieba snownlp requests beautifulsoup4 numpy pillow
```

或使用提供的批处理文件自动安装：
```bash
启动应用.bat
```

## 🚀 运行方式

### 方法1：使用批处理文件（推荐）
双击运行 `启动应用.bat`

### 方法2：命令行运行
```bash
streamlit run app.py
```

浏览器会自动打开 http://localhost:8501

## 📁 文件说明

- `app.py` - Streamlit主应用
- `news_with_keywords.csv` - 清洗后的新闻数据（含关键词）
- `gzdaily_spider.py` - 新闻爬虫脚本
- `extract_hot_topics.py` - 热点话题提取脚本
- `extract_keywords_for_each_news.py` - 关键词提取脚本
- `启动应用.bat` - Windows一键启动脚本

## 💡 使用提示

1. **首次加载**：系统会自动计算所有新闻的情感得分，可能需要几分钟时间
2. **月份选择**：在左侧边栏选择要分析的月份
3. **搜索新闻**：在搜索框输入关键词，实时筛选相关标题
4. **查看正文**：点击"查看正文"按钮展开阅读完整内容
5. **数据更新**：点击"从官网更新数据"获取最新新闻

## 🔧 技术栈

- **前端框架**: Streamlit
- **数据处理**: Pandas, NumPy
- **中文分词**: Jieba
- **情感分析**: SnowNLP
- **可视化**: Matplotlib, WordCloud
- **网络爬虫**: Requests, BeautifulSoup4

## ⚙️ 配置说明

### 停用词表
内置279个停用词，包括：
- 地域名称（广州、广东等）
- 媒体词汇（记者、新华社等）
- 领导人姓名
- 通用无意义词
- 时间词

### 自定义词典
内置50+专有名词，确保分词准确性：
- 体育赛事（全运会、亚运会等）
- 区域发展（粤港澳大湾区等）
- 会议展会（广交会、两会等）
- 科技产业（人工智能、5G等）

### 中文字体
自动检测Windows系统字体：
1. Microsoft YaHei (msyh.ttc)
2. SimHei (simhei.ttf)
3. SimSun (simsun.ttc)

## 📝 注意事项

1. **情感分析缓存**：首次计算后会保存到CSV，后续直接读取
2. **词云生成**：使用缓存机制，相同文本不重复生成
3. **数据更新**：爬虫可能需要较长时间，请耐心等待
4. **内存占用**：6700+条新闻，建议至少4GB内存

## 🐛 常见问题

**Q: 词云中文显示为方框？**
A: 检查系统是否有中文字体，或手动指定字体路径

**Q: 情感分析很慢？**
A: 首次计算需要时间，之后会从缓存读取

**Q: 搜索没有结果？**
A: 检查关键词是否正确，搜索是大小写不敏感的

**Q: 数据更新失败？**
A: 可能是网站反爬，请稍后重试或手动运行爬虫
